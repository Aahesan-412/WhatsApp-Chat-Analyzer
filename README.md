
📊 WhatsApp Chat Analyzer

The WhatsApp Chat Analyzer is a Python-based data analysis project that helps users explore and understand patterns in WhatsApp conversations. By uploading an exported WhatsApp chat text file, the application processes the data and generates useful insights such as message statistics, user activity, word frequency, and emoji usage.

This project is built using Python and Streamlit, allowing users to interactively analyze chat data through a simple web interface.

---

🚀 Features

- 📁 Upload exported WhatsApp chat file
- 📊 View total messages, words, media files, and links shared
- 👤 Identify the most active users in the chat
- 📈 Analyze message activity over time
- ☁️ Generate a Word Cloud of frequently used words
- 😀 Emoji usage analysis
- 📅 Monthly and daily chat activity visualization

These features help users understand communication behavior and engagement within a WhatsApp group or individual chat.

---

⚙️ How It Works

1. Export your WhatsApp chat as a ".txt" file.
2. Upload the chat file using the Streamlit application.
3. The application preprocesses the chat text using Python.
4. The data is structured into a Pandas DataFrame.
5. Visualizations and statistics are generated to analyze chat behavior.
